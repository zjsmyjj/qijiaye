# Projects

## LLM-Assisted Health Message Pretesting

This project evaluates whether large language models can simulate human responses to visual health messages. Using anti-smoking messages as a case study, I compare LLM-based evaluations with human pretesting data to assess replication, bias, and practical usefulness.

## LLM-Generated Message Stimuli

This line of work develops a framework for generating and validating AI-produced experimental stimuli for communication research. The goal is to improve scalability, transparency, and reproducibility in theory-driven experiments.

## Emotional Flow in Health Campaigns

This project examines whether different sequences of emotional appeals, such as fear followed by hope, shape behavioral intention and behavior differently over time in a longitudinal intervention.
